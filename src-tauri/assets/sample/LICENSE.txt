Provenance
----------
This EPUB is ORIGINAL work authored specifically for the Pillowtome project by
the project itself. It was generated deterministically by the auditable,
stdlib-only script `gen_sample.py` in this directory (no third-party content,
no downloaded/copyrighted book). Because it is self-authored, it is guaranteed
DRM-free and freely redistributable, satisfying FND-01/FND-02's need for a
bundled sample that ships inside the app.

It doubles as an early Simplified-Chinese typography smoke test: the prose is
real CJK text with paragraphs, punctuation, and headings, long enough to
paginate across multiple pages so the page-turn verification has content to
advance through.
