# Phase 3 — UI Design Contract

> Visual and interaction contract for CJK Typography Differentiation (CJK-01..05).
> Generated by gsd-ui-researcher; verified by gsd-ui-checker.
> **Extends Phase 2 UI-SPEC** — inherit spacing, typography, color, chrome layout, and Aa sheet shell unless this document overrides.
> Tone remains **极简纸感** (minimal paper-feel): low-contrast chrome; reading surface is primary. CJK controls are quiet preferences, not marketing chrome.

---

| Property | Value |

## Spacing Scale

**Inherited from Phase 2** (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, info icon gap |
| sm | 8px | Compact row spacing within a toggle row |
| md | 16px | Default element spacing, sheet section padding-x |
| lg | 24px | Section padding, default page margins |
| xl | 32px | Gaps between major sheet sections (`gap-8`) |
| 2xl | 48px | Toolbar height |
| 3xl | 64px | Page-level (home only) |

**Exceptions (inherited + Phase 3 additions):**
- Touch targets ≥ **44×44px** for toolbar icons, switches, and info buttons
- Reader toolbar height: **48px** fixed
- Bottom sheet drag handle hit area: **24px** tall
- Progress bar thickness: **2px**
- Immersive tap zones: center **34%**, left/right **33%** (unchanged)
- **CJK toggle row min-height:** **48px** (label + switch + info comfortably fit)
- **Info popover max-width:** **280px** (phone) / **320px** (desktop); padding **md** (16px)

## Typography

### Chrome / UI only (Geist Variable) — inherited

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Caption / meta | 12px | 400 | 1.4 | Progress %, helper text, info popover body |
| Body / label | 14px | 400 | 1.5 | Toggle labels (标点挤压 / 盘古之白 / 禁则), sheet labels |
| Control title | 16px | 600 | 1.4 | Section titles including **中文排版** |
| Sheet heading | 18px | 600 | 1.3 | 显示设置, 目录, 搜索 |

Weights declared: **400** and **600** only.

### Reading body (foliate-js renderer CSS — NOT shell)

| Property | Default | Range | Step | Phase 3 note |
|----------|---------|-------|------|--------------|
| Font size | 18px | 12–32px | 1 | Unchanged |
| Line-height | **1.75** | 1.2–2.4 | 0.05 | **D-41: do not change default** |
| Margins | 24px | 8–48px | 4 | Unchanged |
| First-line indent | **2em** on body `p` | fixed (not user slider) | — | **CJK-04 / D-40** always on; not a toggle |
| Font family | PillowBundledCJK → system CJK stack | custom first when selected | — | **CJK-05 / D-47** |

**Default font stack after Phase 3:**
```
"PillowCustom-{id}"? , "PillowBundledCJK", system-ui, "PingFang SC",
"Hiragino Sans GB", "Microsoft YaHei", "Noto Sans CJK SC", "Noto Sans CJK TC",
sans-serif
```
- Custom face (if active) stays first
- Bundled Noto Sans CJK (SC+TC) is next so incomplete custom faces still fall back for CJK glyphs
- Never use Geist for book body text

**Invisible reader CSS (no chrome UI):**
- `text-indent: 2em` on body paragraphs; `0` on headings, blockquotes, lists, tables, pre
- Full-width quotes participate in kinsoku/spacing at CSS level only — **no character rewrite** (D-42)
- Feature CSS gated by toggles + `CSS.supports` (D-35); silent degrade (D-38)

## Layout Contract

### Reader root / toolbar / immersive / TOC / Search

**Unchanged from Phase 2.** Phase 3 does not add top-level panels, toolbar buttons, or immersive chrome.

### Bottom sheet — 显示设置 (Aa) — **extended**

Entry, max height, handle, scroll body rules: **same as P2**.

**Sections (top → bottom) after Phase 3:**

| # | Section | Source |
|---|---------|--------|
| 1 | 阅读模式 | P2 |
| 2 | 主题 | P2 |
| **3** | **中文排版** | **P3 NEW (D-31)** |
| 4 | 字体 | P2 |
| 5 | 字号 | P2 |
| 6 | 行距 | P2 |
| 7 | 边距 | P2 |

**Placement rationale:** after 主题 / before 字体 so typography features sit with theme; face picker stays with import/remove. Matches 03-RESEARCH Pattern 7.

Section spacing: **xl (32px / `gap-8`)** between sections; **md (16px)** within. Optional `Separator` before 字号 (existing P2 pattern) may remain; do **not** require a separator around 中文排版 unless visual density needs it.

### Section: 中文排版 (NEW)

```
┌─ 中文排版 ─────────────────────────────┐
│ 标点挤压              [i]   [====ON]   │
│ 盘古之白              [i]   [====ON]   │
│ 禁则                  [i]   [====ON]   │
└────────────────────────────────────────┘
```

| Property | Spec |
|----------|------|
| Title | `中文排版` — control title role (16px / 600) |
| Rows | Exactly **3**, independent, no master “优化中文排版” switch (deferred) |
| Row layout | Horizontal: **label (flex 1)** → **info button** → **Switch** |
| Label | 14px / 400; single line; truncate with ellipsis only if forced (labels are short) |
| Info button | lucide `CircleHelp` or `Info`; 44×44 min hit; `aria-label` = `关于{label}` |
| Switch | shadcn `Switch`; thumb track; checked = accent; min hit via wrapper if needed |
| Defaults | All **ON** (`true`) — D-32 |
| Apply | Live on toggle (same `onPrefsChange` / D-22 debounce as other prefs) |
| Persistence | Global `reading_prefs` only — D-34; no per-book UI |
| Disabled state | Switches stay **enabled** even when engine cannot honor the feature (D-38 silent degrade). Do not grey-out or lock toggles based on feature-detect. |
| Capability badge | **None** — no “当前设备不支持” chip under rows |
| Upgrade wall | **Forbidden** — no mandatory WebView upgrade dialog (D-38) |

### Info affordance (D-33)

| Property | Spec |
|----------|------|
| Trigger | Icon button on each row |
| Presentation | **Popover** anchored to the info button (preferred). On very small width or if popover fails a11y on Android, fall back to a short bottom sheet / dialog with the same copy. |
| Title | Same as toggle label (14–16px / 600) |
| Body | Plain-language 简体中文, 1–2 short sentences (caption 12px or body 14px / 400 / 1.5) |
| Close | Tap outside, Escape, or explicit 关闭 if dialog fallback |
| Content | Static help only — no links to external upgrade guides required |
| Tone | Calm, short, non-academic (CONTEXT discretion) |

### Reading surface (invisible contract)

| Concern | UI implication |
|---------|----------------|
| CJK CSS inject | No loading spinner for CSS re-apply; page reflows in place |
| Autospace shim | No user-visible “shim active” indicator |
| Font materialize | First open may briefly use system stack then swap to bundled face (`font-display: swap`); no mandatory blocking splash for fonts |
| Golden / coverage | Dev/CI only — **no in-app QA UI** in P3 |

### States (Phase 3 delta)

| State | Visual |
|-------|--------|
| Toggle ON | Switch checked (accent track) |
| Toggle OFF | Switch unchecked (muted track) |
| Info open | Popover above/below row; dimmed focus trap per Radix defaults |
| Engine unsupported | **Identical UI to supported** — feature simply has no/weak visual effect on the page |
| Prefs load fail | Soft-fail to defaults with all three CJK toggles ON (same as P2 prefs soft-fail) |
| Empty / error for CJK section | **None** — section always has three rows; no empty state |
| Destructive in this phase | **None** for CJK controls |

## Interaction Contract

### Requirement → UI mapping

| Req | Interaction |
|-----|-------------|
| CJK-01 | 显示设置 → 中文排版 → 标点挤压 Switch; live CSS `text-spacing-trim` when caps allow |
| CJK-02 | 盘古之白 Switch; native autospace or non-mutating shim; never permanent text rewrite |
| CJK-03 | 禁则 Switch; `line-break: strict` path when caps allow |
| CJK-04 | No extra controls — first-line indent + lh default applied automatically via `buildReadingCss` |
| CJK-05 | No font picker row for bundled Noto — bundled face is automatic in stack; custom fonts remain P2 字体 list |

### Gesture / focus priority inside Aa sheet

1. Sheet scroll (vertical) takes priority over row swipe
2. Switch thumb drag / click toggles feature
3. Info button opens help without toggling the switch
4. Closing info does not change switch state

### Persistence

- Three booleans in global `reading_prefs` (SCHEMA_V3 append), defaults `1` / `true`
- Debounced save with existing P2 pattern (`PREFS_SAVE_DEBOUNCE_MS`)
- No sync UI (Phase 7)

### Accessibility (minimum)

- All three switches: `aria-label` matching visible label (标点挤压 / 盘古之白 / 禁则)
- Info buttons: `aria-label` `关于标点挤压` / `关于盘古之白` / `关于禁则`
- Popover content announced when opened (Radix Popover / Dialog semantics)
- Do not rely on color alone for ON/OFF — Switch geometry + position
- Touch targets ≥ 44px
- Sheet body keeps `min-h-0 overflow-y-auto` (Android WebView scroll lesson)
