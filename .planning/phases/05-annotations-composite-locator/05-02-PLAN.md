This is NEW shared code — do NOT re-implement CFI parsing (import `cfiToRange` from `./scroll-cfi`) or 简繁 conversion (import `convertText` from `./cjk-convert-shim`).

    Implement the three tiers as directive prose (RESEARCH Self-Healing Resolver 契约, silent stepwise degradation):
    1. CFI tier: if `isRealCfi(cfi)`, `cfiToRange(doc, cfi)`; if the range has non-empty client rects, return `{ range }`.
    2. text_context tier: build a normalized haystack = `convertText(doc.body.textContent, 't2s')` with whitespace collapsed; build a normalized needle = `convertText(text_exact, 't2s')`. Exact-match the needle; if multiple hits, pick the occurrence whose surrounding text best matches normalized `text_pre` (as a suffix ending at the hit) + `text_post` (as a prefix following the hit). If only pre/post exist (zero-length exact, e.g. a reading-position anchor), locate by the `text_pre`+`text_post` window. Map the winning character offset back to a DOM Range with a `SHOW_TEXT` TreeWalker accumulating text-node lengths (same NodeFilter pattern as `getVisibleRange`). On a hit return `{ range, healed:true }`. Include the degraded-window partial-match branch (`text_pre.slice(-8) + text_exact.slice(0,8)`) before giving up.
    3. fraction tier: if `progress_fraction != null`, return `{ fractionTarget: progress_fraction }` — the CALLER lands it on the nearest paragraph boundary; the resolver never scrolls. Else return null.

    Note: `text search` operates on the passed-in `doc` (the section or book document the caller supplies); this task does not itself walk the whole book — the caller decides section-first vs whole-book scope. Word-boundary disambiguation may use `Intl.Segmenter` (already used in cjk-content-transform.ts) but is optional; exact string match + pre/post is the baseline.

    Create `anchor-resolver.test.ts`: construct small `Document` fixtures (jsdom via vitest) and assert each tier — CFI hit returns a range; a broken CFI with a unique text_exact returns `{healed:true}`; two identical excerpts disambiguate by pre/post; a Traditional-form stored needle resolves against a Simplified haystack (and vice versa) proving t2s normalization; a no-match with progress_fraction returns `{fractionTarget}`; total failure returns null.
