<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Stored annotation note/excerpt → React render | Note text (user) + text_exact (book) are shown in the bubble/sheets |
| Clipboard (复制 action) | Selection text copied out |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-10 | Tampering / Elevation | stored XSS via note/excerpt rendered in React | mitigate | Render note + excerpt as React text nodes (default escaping); zero `dangerouslySetInnerHTML` in any Phase 5 component (grep-gated) |
| T-05-11 | Info Disclosure | 复制 writes selection to clipboard | accept | User-initiated copy of content they are already viewing; no elevation, no cross-origin exfil |
| T-05-12 | Tampering | full-screen pointer-capture layer swallowing scroll/selection | mitigate | Bubble is the only pointer-events:auto element (D-74/gate #1); sheets use the TocSheet touch-gate-safe body; verified finger-swipe in 05-05 |
| T-05-SC | Tampering | shadcn textarea / npm installs | accept | textarea is an official shadcn source copy (registries:{} empty) — no third-party registry, no new npm dependency |
</threat_model>
