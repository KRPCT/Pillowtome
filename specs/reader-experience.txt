;===============================================================
; 枕籍阅读体验验收规格（BDD / ATDD）
; 设备基线：雷电模拟器 emulator-5554（Android 9 / WebView 91），
; 低端环境即压力基线。规格只描述外部可观察行为与可测量阈值。
;===============================================================

;===============================================================
; 字体：内置中文字体全部可用
;===============================================================
GIVEN 应用已安装并完成首次启动.
WHEN 打开任意一本 EPUB 书籍.
THEN 正文以当前所选字体渲染，不再出现字体解码失败.
AND 打开书籍后 3 秒内字形稳定，无二次替换闪烁.

GIVEN 阅读器的显示设置已打开.
WHEN 用户把字体切换为「思源宋体」.
THEN 预览区文字以宋体字形显示.
AND 正文重新打开后以宋体字形渲染.

GIVEN 显示设置已打开.
WHEN 用户调整字号、行距或主题.
THEN 预览区在 1 秒内反映新的字号、行距与配色.

;===============================================================
; 进度条与目录
;===============================================================
GIVEN 已打开一本多章节书籍，底部进度条可见.
WHEN 用户把进度条拖到约 50% 处.
THEN 阅读位置落在全书 40%–60% 区间内.
AND 底部百分比与章节名同步更新.

GIVEN 已打开一本多章节书籍.
WHEN 用户打开目录并点击「第二章」.
THEN 正文跳到第二章起始位置.
AND 底部章节名显示为第二章.

;===============================================================
; 翻页动画与手感
;===============================================================
GIVEN 分页模式下已打开一本书.
WHEN 用户点按右半屏翻到下一页.
THEN 页面以平滑滑动完成过渡，时长在 150–500ms 之间.
AND 过渡期间不出现白屏、闪黑或整页跳变.

GIVEN 系统开启了「减弱动态效果」.
WHEN 用户翻页.
THEN 不播放滑动动画，直接到位.

;===============================================================
; 性能：静置与持续使用
;===============================================================
GIVEN 阅读器已打开一本书.
WHEN 静置 30 秒.
THEN 日志中新增 0 条 ResizeObserver loop 错误.
AND 应用无持续重排（CPU 占用回落到接近 0）.

;===============================================================
; 压力验收（阈值即验收线）
;===============================================================
GIVEN 一本 50 章的 EPUB.
WHEN 以约每秒 2 次的速度连续翻页 100 次.
THEN 翻页响应时间 P95 ≤ 800ms.
AND 无崩溃、无白屏、无「文件已损坏」错误页.

GIVEN 一本 50 章的 EPUB.
WHEN 反复打开再返回书库 20 次.
THEN JS 堆内存增长 ≤ 50%.
AND 每次都能正常打开正文.

GIVEN 目录包含 50 个章节.
WHEN 随机点击其中 10 个章节.
THEN 每次都落到目标章节起始处，误差不超过一章.
;===============================================================
