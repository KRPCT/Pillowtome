    **Signature (discretion within D-35):** Prefer adding optional 4th argument `caps: CjkCssCaps` (default all-false or detect-less safe defaults for unit tests). Do not invent a second exported CSS builder.

    **Always emit CJK-04 defaults (not toggles) with !important (D-39, D-40, D-41):**
    - `body p { text-indent: 2em !important; }` (body paragraphs only).
    - Zero indent on non-body: `body h1–h6`, `body blockquote`, `body pre`, `body li`, `body td`, `body th` (and similar non-body blocks if already targeted — keep list tight).
    - Keep existing body/p line-height from prefs; **do not** change DEFAULT_PREFS.lineHeight 1.75 (D-41).
    - **Never** rewrite quote characters in CSS content or via any string transform (D-42). Full-width quotes participate only via kinsoku/line-break CSS.

    **CJK-01 标点挤压 (D-35, D-36 CSS-primary):**
    - If `prefs.cjkPunctTrim && caps.textSpacingTrim` → `html, body { text-spacing-trim: normal !important; }` (or equivalent selector consistent with existing builder root).
    - If toggle OFF and caps support → prefer `text-spacing-trim: space-all` or omit per engine-safe OFF path; document choice in comment.
    - If unsupported → emit nothing for trim (silent degrade D-38). **No JS punctuation rewriter.**

    **CJK-02 盘古之白 CSS path:**
    - If `prefs.cjkAutospace && caps.textAutospace` → `text-autospace: normal !important` (ideograph-alpha + ideograph-numeric).
    - If `!prefs.cjkAutospace` and caps → `text-autospace: no-autospace !important` when supported.
    - If `prefs.cjkAutospace && !caps.textAutospace` → do not emit native property; FoliateView/stream will enable shim (Task 2). Optionally export a pure helper `shouldInstallAutospaceShim(prefs, caps): boolean` from this module or cjk-autospace-shim.

    **CJK-03 禁则:**
    - If `prefs.cjkKinsoku && caps.lineBreakStrict` → `line-break: strict !important; word-break: normal !important; overflow-wrap: break-word !important`.
    - If kinsoku OFF → `line-break: auto !important` when useful.
    - **Hard ban:** never emit `word-break: break-all`.

    **Font faces:** Concatenate existing `fontFaceCss` argument as today; callers in FoliateView should pass `buildFontFaceCss(...) + buildBundledCjkFontFaceCss()` so 03-03 can fill bundled faces without rewriting FoliateView again.

    **Unit tests (extend apply-reading-styles.test.ts):**
    - Defaults: three CJK flags true; lineHeight 1.75.
    - Indent: CSS contains `text-indent: 2em` and zero-indent for `h1` (or headings selector).
    - Emit trim/autospace/line-break only when toggle+caps true.
    - OFF paths do not leave strict/normal incorrectly when toggles false (assert expected OFF tokens).
    - Never contains `break-all`.
    - Never contains quote-rewriting content properties for `"` → `「`.
