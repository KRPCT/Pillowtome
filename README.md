枕籍是一款桌面与手机通用的电子书阅读器：同一套核心程序同时编译到 Windows 和安卓，书籍渲染交给系统自带的网页组件——那是现成最强的中文排版引擎。

一句话：**中文阅读质量是硬指标，不为「先英文跑通」让路。**

## 特性

- **优化排版**：中文与西文、数字之间自动留出间距，老旧系统组件也有内置方案兜底；标点避头尾、词语不拆行、两端对齐。
- **简繁转换**：基于 OpenCC 的显示层转换，选用接近等长的替换方案，阅读位置与进度不受影响，随时切换。
- **内置字体**：应用自带思源宋体可变字体，全书渲染稳定，阅读中字体不会「现场变脸」；也可以选用书内字体与系统字体。
- **两种翻页**：分页与连续滚动两种模式，切换模式、切换简繁、调整字号后都能回到原位置。
- **批注笔记**：划线高亮、批注编辑、选词气泡；批注锚定在文本内容上，书籍重新导入后仍能找回位置。
- **书库管理**：支持 EPUB 与 TXT 导入、文件夹扫描、封面提取，可按最近阅读、书名、作者、进度排序，按在读状态筛选。
- **书内索引**：全文搜索（中文友好）、目录跳转、阅读进度一目了然。
- **自托管同步**：书籍、进度、批注经你自己的网络硬盘互通；冲突自动调和，密码存进系统钥匙串，应用界面拿不到明文。
- **本地优先**：无账号、无广告、无追踪。不开启同步也能完整使用全部功能。

## 下载安装

前往 [Releases](https://github.com/KRPCT/Pillowtome/releases/latest) 下载对应平台安装包：

| 平台 | 安装包 |
|---|---|
| Windows | `Pillowtome_*_x64-setup.exe`（安装程序）或 `Pillowtome_*_x64_en-US.msi`（批量部署用） |
| 安卓 | `pillowtome-*-universal-release.apk`（通用包，覆盖主流处理器架构） |

**首次运行提示**：当前版本未做商业代码签名。Windows 请在安全防护提示中选「更多信息 → 仍要运行」；安卓需允许「安装未知来源应用」。安装包使用自签名证书，后续版本覆盖安装时请保留同一签名。

## 中文排版

枕籍的差异化不在功能清单长度，而在每一页中文的渲染质量：

- **混排间距**：中文与西文字母、数字之间自动留出呼吸空间；系统组件不支持时自动启用内置方案，两端体验一致。
- **标点纪律**：避头尾、悬挂标点、两端对齐共同保证页缘整齐。
- **字体稳定**：可变字重思源宋体随应用打包并注入阅读视图，不受系统字体漂移影响。
- **简繁随心**：只在显示层转换，不动源文件，不丢进度。

## 自托管同步

同步不依赖任何专有云：填一个 WebDAV 地址（坚果云、Nextcloud、群晖均可），书籍、进度、批注都存放在你自己的存储里。登录凭证写入操作系统钥匙串（Windows 凭据管理、安卓系统密钥库），应用界面拿不到明文。

## 技术栈

| 层 | 选型 |
|---|---|
| 应用壳 | Tauri 2（Rust 核心，桌面与安卓同一份代码） |
| 前端 | React 19 + TypeScript（严格模式）+ Vite |
| 渲染内核 | foliate-js（MIT 许可），运行于系统网页组件 |
| 中文能力 | OpenCC 简繁转换，混排间距与标点避头尾均有内置回退方案 |
| 内置字体 | 思源宋体可变字体（随应用打包） |
| 本地存储 | SQLite（书库、进度、批注） |
| 同步 | Rust 直连 WebDAV，凭证交给系统钥匙串 |

## 从源码构建

环境要求：Node 22（长期支持版）、Rust 稳定版、pnpm 10（经 corepack 启用）；构建安卓安装包另需安卓开发套件与原生开发套件。

```bash
corepack enable
pnpm install            # 安装依赖（版本已锁定）
pnpm tauri dev          # 开发模式启动桌面应用
pnpm tauri build        # 打包 Windows 安装包，产物在 src-tauri/target/release/bundle/
pnpm tauri android build --apk   # 构建安卓安装包
```
